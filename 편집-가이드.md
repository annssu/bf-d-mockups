# ✏️ 베프디 웨잇리스트 — 직접 고치기 가이드

코딩 몰라도 글자·줄바꿈·강조 정도는 직접 바꿀 수 있어요.
딱 이 5개만 알면 됩니다.

---

## 🟢 황금 규칙 (이거 하나만 지키면 안 깨져요)

> **`>` 와 `<` 사이에 있는 "글자"만 바꾸세요.**
> `class="..."` 처럼 따옴표 안에 영어로 된 부분은 **디자인 설정**이라 건드리면 모양이 깨져요.

```
<h2 class="text-base font-semibold text-zinc-100">여기 이 글자만 고치기</h2>
                                                  └────── 안전지대 ──────┘
```

파일 안에서 `✏️` 표시를 찾으면, 바로 아래 줄이 고쳐도 되는 곳이에요.

---

## 1. 줄바꿈 (한 줄 내리기)

내리고 싶은 자리에 `<br />` 를 넣으세요.

```
바이브 코딩으로 만든 제품,<br />몇 번이나 돌려보셨어요?
```
→ 화면에선:
```
바이브 코딩으로 만든 제품,
몇 번이나 돌려보셨어요?
```

## 2. 굵게 강조

굵게 하고 싶은 글자를 `<b>` 와 `</b>` 로 감싸기.

```
각자 <b>현업의 진짜 문제</b>를 들고 모여
```
→ "현업의 진짜 문제" 만 굵게.

## 3. 보라색 강조 (A안 기준)

```
<span class="text-violet-400">강조할 글자</span>
```
→ 그 글자만 보라색.  (B안은 보라 대신 그냥 굵게/색 빼고 쓰세요)

## 4. 띄어쓰기 ⚠️ 자주 실수하는 곳

글자 사이는 그냥 **스페이스 한 칸**이면 돼요.
특히 **쉼표(,) 마침표(.) 뒤에는 꼭 한 칸 띄우기!**

```
아닌,계속   ❌  (붙어 보임)
아닌, 계속   ✅
```

## 5. 글자 크기 키우기/줄이기 (살짝 고급)

`text-[34px]` 처럼 적힌 숫자를 바꾸면 글자 크기가 바뀌어요. (px = 픽셀)
숫자 키우면 커지고, 줄이면 작아져요. **숫자만** 바꾸고 나머진 그대로 두세요.

```
text-[34px]   →   text-[40px]   (더 크게)
```

## 💬 나(해니) ↔ 클로드 협업 방법

이게 제일 편해요:

1. 이 파일들을 열어서 `✏️` 표시 근처 글자를 직접 고쳐봐요.
2. 헷갈리거나 모양이 이상해지면, **그 부분을 복사해서** 클로드한테 붙여넣고
   *"여기 이렇게 바꾸고 싶은데 도와줘"* 라고 말하면 돼요.
3. 깨질까 걱정되면 고치기 전에 *"이거 고쳐도 안 깨져?"* 하고 먼저 물어봐도 OK.

> 💡 미리보기는 고친 뒤 브라우저 **새로고침(⌘R)** 하면 바로 보여요.
>    (미리보기 주소: http://127.0.0.1:8765/waitlist-a.html / waitlist-b.html)
